KEY TAKEWAYS FROM ALL LESSONS

What is Computing?

Topic 1

Learning about the Zeroth Generation taught me how computing started with simple mechanical tools like the abacus, eventually evolving into today’s powerful technology. Each era, from vacuum tubes to integrated circuits, built on the last, driving innovation and transforming society. It’s amazing to see how these advancements shaped industries and opened doors for future breakthroughs, reminding me that even small steps can lead to big changes.

Information Processing Cycle

Topic 2

Understanding the Information Processing Cycle really helped me grasp how data turns into meaningful information. It starts with input, where data is collected and validated. Then, the computer processes it, stores it, and finally outputs it in a way that's useful to the user. I also learned that data is just raw facts, while information is processed data that has value for decision-making. This cycle is essential for turning simple data into something actionable, and knowing how each step works makes me appreciate how systems organize and handle information.

Introduction to Computer Systems

Topic 3

I learned a lot about computer systems in this topic! It’s basically about how everything works together: hardware, software, people, and data. Hardware is all the physical stuff like input devices (keyboards), output devices (monitors), and storage. Software tells the computer what to do, and it comes in two types: system software (like the operating system) and application software (like word processors or games). People are the ones who use the computers, and data ware helps manage all the data that gets processed. All these parts work together to make the computer useful and efficient.

Basic Concept of Computer Security

Topic 4

This topic really opened my eyes to the importance of computer security. I learned that securing data is crucial to preventing cybercrimes like identity theft, hacking, and cyberbullying. For example, encrypting sensitive files before sending them is necessary to protect personal data. I also discovered the different types of cybercriminals, such as hackers (both white and black hats) and scammers, who use various methods like phishing to steal information. Understanding the countermeasures like encryption, firewalls, and proxies was also enlightening. These tools are essential in defending against attacks and protecting both personal and business data. Cybersecurity is all about being proactive—whether it's classifying information, securing personal details, or using the right software to prevent attacks.

The Internet and World Wide Web

Topic 5

I learned about that the Internet is a vast network that encompasses various tools and technologies, such as the World Wide Web, e-mail, chat rooms, and file transfer protocols. Cloud computing offers flexible, remote access to data and applications, benefiting businesses by reducing the need for in-house data centers. Machine learning, a branch of artificial intelligence, allows systems to learn and adapt from data, while neural networks improve performance in tasks like speech and image recognition. The Internet of Things connects devices to collect and exchange data, enhancing efficiency in areas like healthcare and manufacturing. Common Internet uses include communication through email and chat, research for a variety of fields, and publishing via the World Wide Web. Social media platforms allow users to share content and engage with others through different forms, such as social networking sites, image and video sharing, and community blogs.

Basic HTML and CSS

Topic 6

In this topic, I learned the fundamentals of HTML and CSS, which are essential for creating and styling web pages. HTML provides the structure of a page using tags like h1, p, and img, which organize content, while CSS is used to style those elements by adjusting their color, size, and layout. I also discovered different ways to apply CSS—inline, internal, and external—and how selectors target specific HTML elements for styling. Additionally, I learned the difference between CSS, which handles the visual aspects, and JavaScript, which adds interactivity. The topic also introduced wireframes and mockups, with wireframes being simple layouts for planning structure and mockups being more detailed visual representations for refining design. This chapter gave me a solid foundation in web development, covering both the structure and appearance of websites.

HIGHLIGHT OF MY FAVORITE LESSON

            Topic 6 HTML and CSS is my favorite topic of the subject so far. I learned HTML and CSS back in senior high school, and I am so happy that I got to learn the basics again. HTML (Hypertext Markup Language) is used to structure web pages with elements represented by tags that define content like headings, paragraphs, and images. Key tags such as html, head, title, and body organize the page, while others like h1 to h6, p, and img structure the content. CSS (Cascading Style Sheets) is used for styling HTML elements, controlling aspects like color, size, and layout, and can be added via inline, internal, or external styles. CSS selectors target specific elements by name, ID, or class to apply styles, while JavaScript adds interactivity and dynamic behavior. Wireframes and mockups are design tools used to plan the layout and visualize the final product, with wireframes focusing on structure and mockups providing a more detailed, polished visual representation.
